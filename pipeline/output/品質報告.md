### [ISSUE-005] ✓ FIXED - 文件 #01 甘特圖 Mermaid 語法錯誤

**狀態：修復完成**

**原始問題：**
- 甘特圖使用 `dateFormat X` 和 `axisFormat D+%s`
- **缺陷：** Mermaid 將相對日期（天數）誤解為毫秒級時間戳
- **現象：** x 軸顯示異常數值（如 D+230400、D+8092800）

**根本原因：**
```
dateFormat X ← 指定毫秒級時間戳格式
但代碼傳入天數 ← Mermaid 直接轉換為毫秒
結果：1 天 → 86,400,000 毫秒 → 顯示為 D+86400000
```

**修復方案：**
- 改用真實日期格式：`dateFormat YYYY-MM-DD`
- 更新 `axisFormat` 為正確格式：`axisFormat %b %d (D+%j)`
- 將所有相對日期轉換為實際日期（基準日期 2026-02-01）

**修復驗證：** ✓ x 軸數值現已正確顯示為實際日期

**影響檔案：** pipeline/output/01_工作計畫書.md（第 299-333 行）
