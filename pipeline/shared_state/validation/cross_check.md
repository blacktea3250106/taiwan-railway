## 三、KPI 數字一致性（以 decisions.yaml 為唯一正確來源）

### 問題 3-1：#01 §4.2 IVR 回應時間拆解與 #03/#10 詳細分配矛盾

**位置：** `01_工作計畫書.md` 第 407 行

> 現有文字：「STT **≤1,500ms** + LLM ≤2,500ms + TTS ≤800ms + 網路/排隊 ≤200ms」（= 5,000ms）

**問題：** #03 §4.1.2 與 #10 ch3 §3.6 的逐步延遲預算為：

| 步驟 | #03 | #10 ch3 |
|------|-----|---------|
| RTP 接收 | ≤10ms | — |
| VAD | ≤200ms | ≤200ms |
| STT | ≤800ms | ≤800ms |
| 意圖辨識 | ≤200ms | ≤200ms |
| 個資遮罩 | ≤50ms | — |
| LLM | ≤2,500ms | ≤2,500ms |
| TTS | ≤800ms | ≤800ms |
| RTP 發送 | ≤50ms | — |
| **合計** | **≤4,610ms**（文件寫 4,600） | **≤4,500ms** |

#01 中「STT ≤1,500ms」應對應 VAD(200) + STT(800) + 意圖(200) = **1,200ms**，非 1,500ms。「網路/排隊 ≤200ms」則應對應 RTP 收(10) + 個資(50) + RTP 發(50) = **110ms**，非 200ms。

**修正 #01 第 407 行：**
> 改為：「VAD+STT+意圖辨識 **≤1,200ms** + LLM ≤2,500ms + TTS ≤800ms + RTP 收發/個資遮罩 **≤110ms**（合計 ≤4,610ms，餘裕約 400ms）」

### 問題 3-2：#03 與 #10 IVR 延遲合計不一致

**位置：**
- `03_程式設計規格報告書.md` 第 374–375 行：「合計 ≤4,600ms…餘裕：400ms」
- `10_程式設計書_ch3.md` 第 238 行：「合計 ≤4,500ms，餘裕 500ms」

**問題：** #03 計入 RTP 收發(10+50ms) 與個資遮罩(50ms) 共 110ms，#10 省略了這些步驟，故差 100ms。

**修正 #10 ch3 第 238 行：**
> 改為：「VAD ≤200ms → STT ≤800ms → 意圖辨識 ≤200ms → **個資遮罩 ≤50ms →** LLM 推論 ≤2,500ms → TTS ≤800ms → 合計 **≤4,600ms**（含 RTP 收發 ≤60ms），餘裕 **400ms** 內完成 5,000ms SLA」

### 問題 3-3：AI-Agent 延遲合計 #07 與 #10 不一致

**位置：**
- `07_軟體使用及維護手冊.md` 第 177 行：「合計延遲 **≤3,100ms**（餘裕 1,900ms）」
- `10_程式設計書_ch4.md` 第 409 行：「合計 **≤2,900ms**，餘裕 2,100ms」

**問題：** #07 包含 API Gateway(100ms) + 意圖辨識(200ms) 但不含 Prompt 組裝與 Citation 抽取；#10 排除 API Gateway 與意圖辨識但包含 Prompt 組裝(50ms) + Citation(100ms)。兩者計入的步驟不同。

**修正方案：** 統一以 #03 §4.2 完整流程為準（API Gateway 100 + 意圖 200 + 個資 50 + RAG 200 + Prompt 50 + LLM 2,500 + Citation 100 = **3,200ms**，餘裕 1,800ms）。

**修正 #07 第 177 行：**
> 改為：「合計延遲 ≤**3,200ms**（SLA 上限 5,000ms，餘裕 **1,800ms**）」

**修正 #10 ch4 第 409 行：**
> 改為：「API Gateway ≤100ms → 意圖辨識 ≤200ms → 個資遮罩 ≤50ms → 向量檢索 ≤200ms → Prompt 組裝 ≤50ms → LLM 推論 ≤2,500ms → Citation 抽取 ≤100ms → 合計 **≤3,200ms**，餘裕 **1,800ms** 內完成 5,000ms SLA」

- 「Active-Active」
- 「Active-Active + Failover」
- 「Active-Active + Failover 架構」

低優先級，不影響驗收。
