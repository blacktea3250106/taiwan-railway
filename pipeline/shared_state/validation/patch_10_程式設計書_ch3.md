### 修正 1：延遲預算分配與 #03 §4.1.2 不一致（漏列個資遮罩與 RTP 封裝階段）
位置：約第 238 行 / §3.6 模組間介面總覽
原文：<<<
VAD ≤200ms → STT ≤800ms → 意圖辨識 ≤200ms → LLM 推論 ≤2,500ms → TTS ≤800ms → 合計 ≤4,500ms，餘裕 500ms 內完成 5,000ms SLA
>>>
改為：<<<
VAD ≤200ms → STT ≤800ms → 意圖辨識 ≤200ms → 個資遮罩 ≤50ms → LLM 推論 ≤2,500ms → TTS ≤800ms → RTP 封裝 ≤50ms → 合計 ≤4,600ms，餘裕 400ms 內完成 5,000ms SLA
>>>

**說明：** #03 §4.1.2 的完整 pipeline 包含 8 個階段，原文漏列「個資遮罩 ≤50ms」與「RTP 封裝 ≤50ms」兩階段，導致合計少 100ms、餘裕多 100ms，與權威規格文件不一致。
