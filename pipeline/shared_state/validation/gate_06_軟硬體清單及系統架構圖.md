**附帶風險提示（非直接違規但須關注）：**

- **Quectel（§6.2.1 附註）**：SMS 裝置 LTE 模組候選中列出 Quectel，該公司為中國品牌（上海移遠通信），即使標註「台灣產線模組」，品牌本身仍屬大陸廠牌，依契約§8(24)應排除。建議從候選清單移除 Quectel，僅保留 Sierra Wireless（加拿大）及 u-blox（瑞士）。
- **Grafana / Loki（AGPL-3.0）**：AGPL 授權要求網路互動時提供原始碼，雖本案為內部系統風險低，但建議於授權合規聲明中加註 AGPL 使用範圍與義務說明。
